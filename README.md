🎨 Hirst Painting Generator using Python Turtle

📌 Project Overview

This project recreates a Damien Hirst-inspired dot painting using Python's Turtle graphics library. The program draws a 10×10 grid of colorful dots, with each dot's color randomly selected from a predefined color palette.

The artwork is inspired by Damien Hirst's famous spot paintings, which consist of evenly spaced colored dots arranged in a grid pattern.

🚀 Features

Generates a 10×10 grid of dots.

Uses random colors for each dot.

Implements Python Turtle graphics for visualization.

Uses RGB color mode for a wider range of colors.

Automatically creates a colorful and unique painting every time it runs.

🛠 Technologies Used

Python 3

Turtle Graphics

Random Module

📂 How It Works

Set Turtle color mode to RGB.

Store a list of RGB color tuples.

Create a Turtle object.

Draw colored dots across a row.

Move to the next row after completing a line.

Repeat until a 10×10 grid is created.

Keep the window open until the user clicks.

▶️ Running the Project

Install Python

Make sure Python 3 is installed on your system.

Run the Program
python main.py
